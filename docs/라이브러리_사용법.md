# 라이브러리 사용법

## 공통 경로 별칭

- `@/` → `src/`
- 사용 예: `import { cn } from '@/lib/utils'`

## axios

- 공통 인스턴스: `src/lib/api.js`
- 환경 변수: `VITE_API_BASE_URL`

```js
import { api } from '@/lib/api'

export async function fetchUsers() {
  const { data } = await api.get('/users')
  return data
}
```

## react-router-dom

- 라우터 루트: `src/main.jsx`의 `<BrowserRouter>`
- 라우팅 정의: `src/App.jsx`에서 `<Routes>`/`<Route>`

```jsx
import { Routes, Route } from 'react-router-dom'
import Home from '@/pages/Home'
import About from '@/pages/About'

function App() {
  return (
    <Routes>
      <Route path="/" element={<Home />} />
      <Route path="/about" element={<About />} />
    </Routes>
  )
}
```

## Tailwind CSS

- 설정 파일: `tailwind.config.js`
- 전역 CSS: `src/index.css` (CSS 변수 + base/components/utilities)
- 플러그인: `tailwindcss-animate` (애니메이션 유틸 제공)

```jsx
export function Card() {
  return (
    <div className="rounded-lg border bg-background text-foreground shadow-sm">
      내용
    </div>
  )
}
```

## shadcn/ui

- 설정 파일: `components.json`
- 컴포넌트 추가:

```bash
npx shadcn@latest add <component>
```

- 사용 예(추가 후):

```jsx
import { Button } from '@/components/ui/button'

export function SubmitButton() {
  return <Button>Submit</Button>
}
```

## cn 유틸 (clsx + tailwind-merge)

- 위치: `src/lib/utils.js`
- 조건부 클래스 + Tailwind 충돌 정리

```jsx
import { cn } from '@/lib/utils'

export function Badge({ active }) {
  return (
    <span className={cn('px-2 py-1 text-xs', active && 'bg-primary text-white')}>
      상태
    </span>
  )
}
```

## class-variance-authority (cva)

- 변형(variant) 기반 컴포넌트 스타일 정의

```js
import { cva } from 'class-variance-authority'

export const buttonVariants = cva('inline-flex items-center', {
  variants: {
    variant: {
      default: 'bg-primary text-primary-foreground',
      ghost: 'bg-transparent',
    },
    size: {
      sm: 'h-8 px-3 text-sm',
      lg: 'h-10 px-4',
    },
  },
  defaultVariants: {
    variant: 'default',
    size: 'sm',
  },
})
```

## @radix-ui/react-slot

- 컴포넌트의 `asChild` 패턴에 사용 (shadcn/ui 내부에서 자주 사용)

```jsx
import { Slot } from '@radix-ui/react-slot'

export function LinkButton({ asChild, ...props }) {
  const Comp = asChild ? Slot : 'button'
  return <Comp {...props} />
}
```

## zustand

- 전역 상태 관리

```js
import { create } from 'zustand'

export const useCounterStore = create((set) => ({
  count: 0,
  inc: () => set((state) => ({ count: state.count + 1 })),
}))
```

```jsx
import { useCounterStore } from '@/stores/useCounterStore'

export function Counter() {
  const { count, inc } = useCounterStore()
  return (
    <button onClick={inc} className="px-3 py-2">
      {count}
    </button>
  )
}
```
