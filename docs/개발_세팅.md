# 개발 세팅 정리

## 설치된 라이브러리

- axios: API 요청 인스턴스 템플릿을 `src/lib/api.js`에 추가
- react-router-dom: 라우팅 구성 (`src/main.jsx`, `src/App.jsx`)
- tailwindcss: 유틸리티 CSS 기본 설정 (`tailwind.config.js`, `postcss.config.js`)
- shadcn/ui: 컴포넌트 사용 기반 설정 (`components.json`)

## 주요 설정 파일

- Tailwind 설정: `tailwind.config.js`
- PostCSS 설정: `postcss.config.js`
- shadcn/ui 설정: `components.json`
- 경로 별칭: `jsconfig.json` + `vite.config.js`에서 `@/` → `src/`
- 전역 스타일: `src/index.css` (CSS 변수 + Tailwind base/components/utilities)

## shadcn/ui 컴포넌트 추가

```bash
npx shadcn@latest add <component>
```

## 실행/검증

```bash
npm run dev
npm run lint
npm run build
```

## 환경 변수

- API 기본 URL: `VITE_API_BASE_URL`
- GA4 측정 ID: `VITE_GA_ID`
